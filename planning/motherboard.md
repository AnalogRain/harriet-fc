# Harriet

- 1 Harriet CPU
- 2 MHz clock
- 1 Memory Controller
 - Includes up to 256 banks, switched via the `0xFFFF` address
 - All banks have the first 8KB of RAM in common, drivers go here
- 64 KiB of RAM
- 2 floppy disk controllers
 - 3.5 inch floppy disks
- keyboard controller
- ROM w/ BIOS
